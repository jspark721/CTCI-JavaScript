# Binary Numbers
base 2 numbers, computers don't have digits with ten possible values that we use **base 10** which is decimal, base 2 is called **binary**

### The places in binary (base 2) are sequential powers of 2:
- 2^0 = 1
- 2^1 = 2
- 2^2 = 4
- 2^3 = 8
- 2^4 = 16
- etc...
